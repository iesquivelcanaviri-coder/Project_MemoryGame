JavaScript Memory Game

Demonstrating DOM manipulation, state management, and event handling.

How to Play

  • You may flip two cards per turn.
  • Matched cards remain face-up.
  • Unmatched cards flip back automatically.
  • Each pair of flips counts as one move.
  • The timer stops when all pairs are matched.
