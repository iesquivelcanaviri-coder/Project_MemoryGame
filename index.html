JavaScript Memory Game

A beginner-friendly memory game built to learn HTML, CSS and JavaScript.

How to Play

  • You may flip two cards per turn.
  • Cards that match stay face-up.
  • Cards that do not match flip back automatically.
  • Each pair of flips counts as one move.
  • The timer stops when all pairs are matched.
