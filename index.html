JavaScript Memory Game

A beginner-friendly memory game built to practise HTML, CSS and JavaScript.

How to Play

  • You may flip two cards per turn.
  • Matched cards stay face-up.
  • Unmatched cards flip back automatically.
  • Each pair counts as one move.
  • The timer stops when all pairs are matched.
