# JavaScript Memory Game

This is Version 4 of a memory matching game created as part of a JavaScript module.

## Purpose
This version focuses on:
- Writing beginner-friendly code
- Adding clear comments
- Understanding how HTML, CSS, and JavaScript work together

## Features
- Card matching logic
- Move counter
- Timer
- Difficulty selection
- Dynamic DOM manipulation

## Learning Focus
- DOM selection and events
- Arrays and objects
- Game state management
- CSS Grid layout
- JavaScript functions and conditions

## Hosting
This project is hosted using GitHub Pages.
